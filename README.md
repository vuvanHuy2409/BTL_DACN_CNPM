# Các con của bố đọc cho kĩ vào nha
1.  Tải về lần đầu: Chỉ làm 1 lần duy nhất khi máy chưa có code.

    ```Bash
    git clone https://github.com/vuvanHuy2409/BTL_DACN_CNPM.git
2.  Quy trình Code hàng ngày (Quan trọng)
* Bước 1: Cập nhật code mới (Pull)

    ```Bash
    git pull origin main
(Luôn làm bước này đầu tiên khi bật máy)

*   Bước 2: Code và lưu thay đổi (Commit)

    ```Bash
    git add .
    git commit -m "4/12/2025"
* Bước 3: Đẩy lên (Push)
    ```Bash
    git push origin main
